# Модуль учёта лунных возмущений

## Описание работы

Модуль лунных возмущений моделирует влияние гравитационного поля Луны на орбитальное движение искусственного спутника Земли. Луна, как третье тело в системе «Земля-спутник-Луна», создаёт дополнительные возмущающие ускорения, которые могут существенно влиять на эволюцию орбиты, особенно для высокоорбитальных аппаратов.

В отличие от геопотенциальных возмущений, которые убывают с высотой, лунные возмущения возрастают по мере удаления спутника от Земли и становятся доминирующим фактором для орбит с большой полуосью более 10 000 км.

## Цель исследования

**Цель:** Изучить влияние гравитационного притяжения Луны на орбитальное движение искусственного спутника Земли, разработать методику численного интегрирования уравнений движения с учётом лунных возмущений и проанализировать эволюцию орбитальных элементов.

## Поставленные задачи

1. Реализовать математическую модель возмущающего ускорения от третьего тела (Луны)
2. Разработать алгоритм численного интегрирования дифференциальных уравнений движения
3. Исследовать зависимость возмущений от параметров орбиты спутника
4. Проанализировать эволюцию всех пяти орбитальных элементов (a, e, i, Ω, ω)
5. Провести сравнение возмущённой и невозмущённой (кеплеровской) орбит
6. Выполнить анализ периодических и вековых составляющих возмущений
7. Оценить численную точность интегрирования и выработать рекомендации

## Методика расчёта

### Уравнение движения

Рассмотрим систему трёх тел: Земля (m₁), спутник (m₂) и Луна (m₃). Пренебрегая массой спутника, получаем уравнение движения:

```
r̈ = -μ·r/r³ + a_pert
```

где:
- r — радиус-вектор спутника относительно Земли
- μ = G·m₁ — гравитационный параметр Земли
- a_pert — возмущающее ускорение от Луны

### Возмущающее ускорение от третьего тела

Возмущающее ускорение определяется разностью притяжений Луны к спутнику и к Земле:

```
a_pert = -μₗ · [ρ/ρ³ - r₁₂/r₁₂³]
```

где:
- μₗ = 4902.8 км³/с² — гравитационный параметр Луны
- ρ = r - r₁₂ — вектор от Луны к спутнику
- r₁₂ — вектор от Земли к Луне

**Физический смысл:**
- Первое слагаемое — притяжение спутника Луной
- Второе слагаемое — притяжение Земли Луной (реактивная сила)

### Компоненты ускорения в орбитальной системе

Проекция возмущающего ускорения на орбитальную систему координат:

```
S = F_x · cos θ + F_y · sin θ        (радиальная)
T = -F_x · sin θ + F_y · cos θ       (трансверсальная)
W = F_z                              (бинормальная)
```

где θ — истинная аномалия, F_x, F_y, F_z — проекции a_pert на оси ECI.

### Дифференциальные уравнения в элементах

Для анализа эволюции орбиты используется система дифференциальных уравнений в орбитальных элементах. Аргумент широты u выступает в роли независимой переменной:

#### 1. Уравнение для восходящего узла Ω

```
dΩ/du = (r³ · sin u) / (μ · p · sin i) · W
```

Описывает прецессию плоскости орбиты под действием бинормальной компоненты.

#### 2. Уравнение для наклонения i

```
di/du = (r³ · cos u) / (μ · p) · W
```

Изменение наклонения вызвано также компонентой W.

#### 3. Уравнение для фокального параметра p

```
dp/du = (2 · r³) / μ · T
```

Трансверсальная компонента изменяет размер и форму орбиты.

#### 4. Уравнение для эксцентриситета e

```
de/du = (r²)/(μ · e) · [sin θ · S + cos θ · (1 + r/p) · T + e · (r/p) · W]
```

Все три компоненты влияют на эксцентриситет.

#### 5. Уравнение для аргумента перигея ω

```
dω/du = (r²)/(μ · e) · [cos θ · S + e · sin θ · (1 + r/p) · T - e · (r/p) · cot i · sin u · W]
```

Сложная зависимость от всех компонент, включая сингулярность при i → 0.

### Численное интегрирование

Для решения системы дифференциальных уравнений применяется **метод Рунге-Кутты-Фельдберга 4(5) порядка (RKF45)** с автоматическим выбором шага.

#### Преимущества RKF45:

- **Адаптивный шаг** — автоматически подбирается для обеспечения точности
- **Оценка ошибки** — разница между решениями 4-го и 5-го порядков
- **Эффективность** — минимизация вычислений при сохранении точности

#### Алгоритм интегрирования:

1. Вычисление 6 производных (k₁...k₆) на каждом шаге
2. Получение двух решений: y₄ (4-й порядок) и y₅ (5-й порядок)
3. Оценка относительной ошибки: ε = |y₄ - y₅| / |y_prev|
4. Если ε > tolerance (10⁻¹⁰) — уменьшение шага
5. Если ε < tolerance — увеличение шага (но не более чем в 5 раз)

#### Интегрирование по аргументу широты:

Независимая переменная — аргумент широты u ∈ [u₀, u₀ + 2π].

Связь со временем:

```
t = (u - u₀) / n
```

где n = √(μ/a³) — среднее движение.

### Модель движения Луны

Использузуется упрощённая модель орбиты Луны:

**Элементы орбиты Луны:**
- aₗ = 384 399 км — большая полуось
- eₗ = 0.0549 — эксцентриситет
- iₗ = 5.145° — наклонение к эклиптике
- Ωₗ — долгота восходящего узла (задаётся пользователем)
- uₗ — аргумент широты (задаётся пользователем)

**Положение Луны вычисляется по классической схеме:**

1. Средняя аномалия: Mₗ = nₗ · t + Mₗ₀
2. Эксцентрическая аномалия: решение уравнения Кеплера Eₗ - eₗ·sin Eₗ = Mₗ
3. Истинная аномалия: θₗ = 2·atan2(√(1+eₗ)·sin(Eₗ/2), √(1-eₗ)·cos(Eₗ/2))
4. Расстояние: rₗ = aₗ · (1 - eₗ·cos Eₗ)
5. Координаты в ECI через элементы Ωₗ, iₗ, uₗ = θₗ + ωₗ

### Сравнение с кеплеровской орбитой

Для анализа возмущений выполняется параллельный расчёт невозмущённой орбиты:

**Кеплерова орбита:**
- Все элементы постоянны (a₀, e₀, i₀, Ω₀, ω₀)
- Положение вычисляется по уравнению Кеплера
- Возмущающие ускорения отсутствуют

**Возмущённая орбита:**
- Элементы эволюционируют во времени
- Положение определяется текущими элементами
- Учитываются все возмущения от Луны

**Разница** между орбитами характеризует накопительное влияние возмущений.

## Нюансы реализации

### 1. Модель движения Луны

**Упрощения:**
- Орбита Луны считается фиксированной (не эволюционирует)
- Не учитывается влияние Солнца на лунную орбиту
- Долгосрочные эффекты (прецессия узла Луны) игнорируются

**Обоснование:**
- Время интегрирования обычно ≤ 1 периода спутника
- Для ВНО это несколько часов/суток
- За это время изменения лунной орбиты ничтожны

**Перспективы:**
- Использование эфемерид DE405/DE421 для высокой точности
- Учёт солнечного притяжения для межпланетных миссий

### 2. Проекции ускорений

**Система координат S-T-W:**

- **S (radial)** — направлена по радиус-вектору от Земли
- **T (transverse)** — перпендикулярна S в плоскости орбиты (по направлению движения)
- **W (normal)** — перпендикулярна плоскости орбиты (правая тройка)

**Преобразование из ECI:**

Требуется матрица перехода от ECI к орбитальной системе. Используется последовательность поворотов:

```
ECI → Орбитальная через:
1. Поворот на угол Ω вокруг Z
2. Поворот на угол i вокруг новой X
3. Поворот на угол u вокруг новой Z
4. Поворот на угол θ (истинная аномалия)
```

**Особенности:**
- При e → 0 истинная аномалия не определена
- При i → 0 возникает сингулярность в dω/du
- Введены ограничения: e ≥ 0.001, sin i ≥ 10⁻¹⁰

### 3. Ограничения численного метода

**Физические ограничения:**

- **Эксцентриситет:** 0.001 ≤ e ≤ 0.999
  - Защита от вырождения в прямую
  - Защита от гиперболических орбит
  
- **Фокальный параметр:** p > 0
  - Отрицательный p физически недопустим
  - При нарушении — сброс к начальному значению

- **Наклонение:** 0 ≤ i ≤ π
  - Отражение при выходе за границы
  - Нормализация после каждого шага

**Численная стабильность:**

- **Минимальный шаг:** h_min = (u_final - u₀) / 50000
  - Предотвращает зацикливание
  - Ограничивает время счёта

- **Максимальный шаг:** h_max = (u_final - u₀) / 100
  - Предотвращает потерю точности
  - Гарантирует разрешение быстрых осцилляций

- **Допуск ошибки:** ε = 10⁻¹⁰
  - Баланс между точностью и скоростью
  - Типичное значение для двойной точности

### 4. Обработка сингулярностей

**Круговые орбиты (e → 0):**

Проблема: уравнения для de/du и dω/du содержат деление на e.

Решение:
- Введён порог e_min = 0.001
- При e < e_min используется регуляризация
- Альтернатива: переход к элементам типа эквивалентных

**Полярные и экваториальные орбиты (i → 0, π):**

Проблема: dΩ/du и dω/du содержат деление на sin i.

Решение:
- Порог sin i_min = 10⁻¹⁰
- Для экваториальных орбит Ω не определён
- Для полярных орбит нет особенностей

### 5. Производительность

**Оптимизация:**

- **Предварительный расчёт:** положение Луны вычисляется один раз на точку
- **Кэширование:** тригонометрические функции от постоянных углов
- **Векторизация:** обработка массива точек параллельно

**Время счёта:**
- 100 точек, 1 виток: ~50-200 мс
- 1000 точек, 10 витков: ~1-3 с
- Зависит от сложности орбиты (e, i)

## Интерпретация результатов

### Статистика ускорений

Типичные значения лунных возмущений:

**Для низких орбит (300-1000 км):**
- |a|_max ≈ 10⁻⁷ ... 10⁻⁶ м/с²
- Доминирует компонента W (бинормальная)
- Периодические колебания с периодом ~½ периода орбиты

**Для средних орбит (1000-10000 км):**
- |a|_max ≈ 10⁻⁶ ... 10⁻⁵ м/с²
- Все три компоненты сопоставимы
- Появляются вековые тренды

**Для высоких орбит (>10000 км):**
- |a|_max ≈ 10⁻⁵ ... 10⁻⁴ м/с²
- Сопоставимо с геопотенциальными возмущениями
- Сильная зависимость от взаимного положения с Луной

### Анализ компонент S, T, W

**Радиальная компонента S:**
- Влияет на a и e
- Максимальна в новолуние/полнолуние
- Знак меняется при изменении взаимного положения

**Трансверсальная T:**
- Наиболее сильно влияет на e и ω
- Вызывает вековое изменение большой полуоси
- Зависит от фазы Луны относительно спутника

**Бинормальная W:**
- Доминирует для большинства орбит
- Вызывает прецессию Ω и i
- Особенно важна для солнечно-синхронных орбит

### Эволюция орбитальных элементов

**Большая полуось a:**
- Периодические изменения: Δa/a ~ 10⁻⁶ ... 10⁻⁴
- Вековой тренд обычно отсутствует (среднее за период → 0)
- Амплитуда растёт с высотой орбиты

**Эксцентриситет e:**
- Может испытывать значительные изменения
- Для некоторых орбит — вековой рост (опасно!)
- Циклы Лидова-Козаи при больших i

**Наклонение i:**
- Обычно малые изменения (Δi ~ 10⁻⁴ ... 10⁻² град)
- Зависит от взаимной ориентации орбит
- Важно для долгоживущих КА

**Долгота узла Ω:**
- Прецессия от лунных возмущений
- Сопоставима с прецессией от J₂ для ВНО
- Направление зависит от i

**Аргумент перигея ω:**
- Часто наблюдается вековая скорость
- Циклы для резонансных орбит
- Влияет на высоту в апогее/перигее

### Анализ разницы между возмущённой и невозмущённой орбитами

**Методика сравнения:**

1. В каждый момент времени t вычисляются:
   - Положение r_pert(t) для возмущённой орбиты
   - Положение r_kep(t) для кеплеровской орбиты

2. Разница положений:
   ```
   Δr(t) = |r_pert(t) - r_kep(t)|
   ```

3. Относительная ошибка:
   ```
   δ(t) = Δr(t) / r_kep(t) · 100%
   ```

**Типичные результаты:**

**Низкие орбиты (300-1000 км):**
- Δr_max ~ 10-100 м за 1 виток
- δ ~ 10⁻⁴ ... 10⁻³ %
- Лунные возмущения пренебрежимо малы

**Средние орбиты (1000-10000 км):**
- Δr_max ~ 100 м ... 1 км за 1 виток
- δ ~ 10⁻³ ... 10⁻² %
- Необходимо учитывать для точных задач

**Высокие орбиты (>10000 км):**
- Δr_max ~ 1 ... 100 км за 1 виток
- δ ~ 0.01 ... 1 %
- Лунные возмущения доминируют
- Критично для ГЛОНАСС/GPS/ГСО

### Периодические составляющие

**Частотный анализ:**

Обнаруживаются пики на частотах:

- **ω₀** — орбитальная частота спутника
- **2ω₀** — удвоенная орбитальная частота
- **ω₀ ± ωₗ** — комбинационные частоты
- **ωₗ** — лунная частота (для долгих интервалов)

**Причины:**

- Геометрия взаимного положения
- Проекция лунного ускорения на орбитальные оси
- Резонансы при кратности частот

### Качество численного интегрирования

**Метрики качества:**

1. **Сохранение энергии:**
   ```
   ΔE/E = (a_max - a_min) / a_mean
   ```
   - Для хороших расчётов: < 10⁻⁶
   - Допустимо: < 10⁻⁵
   - Плохо: > 10⁻⁴

2. **Гладкость траектории:**
   - Оценивается по второй разности r(u)
   - Характеризует отсутствие скачков

3. **Оценка ошибки:**
   - Автоматически вычисляется в RKF45
   - Должна быть близка к заданному tolerance

**Рекомендации по улучшению:**

- Уменьшить tolerance (10⁻¹¹ ... 10⁻¹²)
- Увеличить количество точек вывода
- Использовать симлектические интеграторы для долгих интервалов

### Рекомендации по использованию

**Когда учитывать лунные возмущения:**

✅ Высокие орбиты (a > 10 000 км)
✅ Долгосрочное прогнозирование (> 1 суток)
✅ Прецизионные задачи (навигация, геодезия)
✅ Исследование устойчивости орбит

❌ Низкие орбиты (a < 2000 км)
✅ Краткосрочные манёвры (< 1 витка)
❌ Грубые оценки

**Выбор параметров интегрирования:**

**Для быстрой оценки:**
- pointsCount = 50-100
- tolerance = 10⁻⁸
- Время: ~10-50 мс

**Для точного расчёта:**
- pointsCount = 500-1000
- tolerance = 10⁻¹⁰ ... 10⁻¹²
- Время: ~100-500 мс

**Для научных исследований:**
- pointsCount = 1000-5000
- tolerance = 10⁻¹²
- Адаптивный шаг обязателен

### Пример интерпретации результатов

**Входные данные:**
- Орбита: a=26600 км, e=0.001, i=55° (тип GPS)
- Время интегрирования: 12 часов (1 виток)

**Полученные результаты:**
- Δa = -0.15 км (уменьшение)
- Δe = +0.00002 (рост)
- Δi = -0.003° (уменьшение)
- ΔΩ = -0.012° (прецессия назад)
- Δω = +0.025° (прецессия вперёд)
- |a|_max = 3.2×10⁻⁵ м/с²

**Выводы:**

1. **Доминирует бинормальная компонента W** — вызывает прецессию Ω и i
2. **Вековая прецессия узла** — 0.024°/сутки, сопоставимо с J₂
3. **Стабильность эксцентриситета** — малый рост, опасности нет
4. **Точность интегрирования** — ΔE/E = 2×10⁻⁷, отлично
5. **Рекомендация** — учитывать лунные возмущения для навигационных КА

## Заключение

Модуль лунных возмущений предоставляет мощный инструмент для анализа динамики орбит под действием третьего тела. Правильная интерпретация результатов позволяет оптимизировать параметры орбит космических аппаратов, прогнозировать их эволюцию и планировать коррекции.
