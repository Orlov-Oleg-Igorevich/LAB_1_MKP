# Модуль возмущений от нецентральности гравитационного поля Земли

## Описание работы

Модуль геопотенциальных возмущений предназначен для расчёта ускорений, возникающих вследствие отклонения гравитационного поля Земли от сферически симметричного. В отличие от простой модели Ньютона, где Земля считается материальной точкой, реальное гравитационное поле сложнее из-за:

- Сплюснутости Земли у полюсов (экваториальное вздутие)
- Неоднородности распределения масс
- Отклонения формы Земли от идеального эллипсоида

Для математического описания используется разложение гравитационного потенциала в ряд по сферическим функциям, что позволяет учесть различные гармоники поля.

## Цель исследования

**Цель:** Изучить влияние нецентральности гравитационного поля Земли на движение искусственного спутника и научиться вычислять возмущающие ускорения для различных моделей геопотенциала.

## Поставленные задачи

1. Реализовать математическую модель гравитационного потенциала Земли в виде ряда по сферическим гармоникам
2. Вычислить компоненты возмущающего ускорения в сферической системе координат
3. Исследовать зависимость возмущений от высоты орбиты и положения спутника
4. Провести сравнительный анализ упрощённой модели (только J₂) и полной модели
5. Проанализировать вклад различных гармоник в суммарное возмущение
6. Реализовать визуализацию результатов расчётов

## Методика расчёта

### Гравитационный потенциал

Гравитациационный потенциал Земли в точке с сферическими координатами (r, φ, λ) представляется в виде:

```
U(r, φ, λ) = μ/r · [1 + Σ(n=2 to ∞) Σ(k=0 to n) (R₀/r)ⁿ · Pₙₖ(sin φ) · fₙₖ(λ)]
```

где:
- μ — геоцентрическая гравитационная постоянная
- R₀ — экваториальный радиус Земли
- Pₙₖ — присоединённые функции Лежандра
- fₙₖ(λ) — азимутальная зависимость

### Возмущающий потенциал

Возмущающий потенциал U_voz определяется как разность между полным потенциалом и потенциалом сферической Земли:

```
U_voz = U - μ/r
```

### Компоненты ускорения

Возмущающее ускорение вычисляется как градиент возмущающего потенциала. В сферической системе координат получаем три компоненты:

#### Радиальная компонента (S)

Направлена по радиус-вектору от центра Земли:

```
S = ∂U_voz/∂r = μ/R₀² · Σ(n=2 to N) Σ(k=0 to min(K,n)) -(n+1) · (R₀/r)ⁿ⁺² · Pₙₖ(sin φ) · Aₙₖ(λ)
```

где Aₙₖ(λ) = Jₙ·Pₙ₀ при k=0, иначе Cₙₖ·cos(kλ) + Sₙₖ·sin(kλ)

#### Меридиональная компонента (T_φ)

Направлена вдоль меридиана:

```
T_φ = (1/r) · ∂U_voz/∂φ = μ/R₀² · cos φ · Σ(n=2 to N) Σ(k=0 to min(K,n)) (R₀/r)ⁿ⁺² · dPₙₖ/d(sin φ) · Aₙₖ(λ)
```

#### Широтная компонента (T_λ)

Направлена вдоль параллели:

```
T_λ = (1/(r·cos φ)) · ∂U_voz/∂λ = -μ/R₀² · (1/cos φ) · Σ(n=2 to N) Σ(k=1 to min(K,n)) k · (R₀/r)ⁿ⁺² · Pₙₖ(sin φ) · Bₙₖ(λ)
```

где Bₙₖ(λ) = -Cₙₖ·sin(kλ) + Sₙₖ·cos(kλ)

### Переход к орбитальной системе координат (S, T, W)

Для анализа возмущений в орбитальной системе координат выполняется преобразование:

```
S = j_r                    (радиальная)
T = j_φ · cos A + j_λ · sin A  (трансверсальная)
W = j_φ · sin A - j_λ · cos A  (бинормальная)
```

где A — азимут, определяемый из соотношений:

```
sin A = cos i / cos φ
cos A = cos u · sin i / cos φ
```

Здесь i — наклонение орбиты, u — аргумент широты.

### Полное ускорение

Модуль полного возмущающего ускорения:

```
|a| = √(S² + T² + W²)
```

## Нюансы реализации

### 1. Использование сферических гармоник

В проекте реализовано разложение до 21-й гармоники включительно, что обеспечивает высокую точность расчётов:

**Зональные гармоники (k=0):**
- J₂ = 1082.63×10⁻⁵ — основная гармоника (сжатие Земли)
- J₃ = -2.53×10⁻⁶ — «грушевидность» формы
- J₄ = -1.62×10⁻⁶ и выше

**Тессеральные и секториальные гармоники (k>0):**
- C₂₁, S₂₁ — смещение центра масс
- C₂₂, S₂₂ — эллиптичность экватора
- И другие до n=21, k=21

### 2. Обработка коэффициентов J₂ и других

**J₂-доминирование:**
- Коэффициент J₂ на два порядка превышает остальные гармоники
- Реализован специальный режим «J₂-only» для сравнения
- При включении только J₂ расчёт ускоряется в 10-20 раз

**Защита от особенностей:**
- В полярных областях (cos φ → 0) вводятся ограничения
- Используется безопасное деление с порогом 10⁻¹²
- Азимут вычисляется через atan2 для избежания неоднозначностей

### 3. Системы координат

Поддерживается работа в двух системах:

**ECI (Earth-Centered Inertial):**
- Инерциальная система, не вращается с Землёй
- Используется для расчёта орбитальной механики
- Координаты спутника фиксированы относительно звёзд

**ECEF (Earth-Centered Earth-Fixed):**
- Вращается вместе с Землёй
- Используется для учёта тессеральных гармоник
- Требует пересчёта долготы с учётом вращения Земли:
  ```
  λ_ECEF = λ_ECI - ωₑ · t
  ```

### 4. Численное интегрирование уравнения Кеплера

Для определения положения спутника на орбите решается уравнение Кеплера:

```
M = E - e · sin E
```

где M — средняя аномалия, E — эксцентрическая аномалия, e — эксцентриситет.

Используется численный метод с точностью ε = 0.001°.

### 5. Оптимизация вычислений

**Предварительные вычисления:**
- Функции Лежандра Pₙₖ и их производные кэшируются
- Тригонометрические функции вычисляются один раз на точку

**Векторизация:**
- Расчёт ведётся сразу для всех точек орбиты (массив из 100+ точек)
- Минимизация циклов за счёт предварительных вычислений

## Интерпретация результатов

### Типичные значения ускорений

Для низких околоземных орбит (300-1000 км):

- **Полное ускорение:** 10⁻⁵ ... 10⁻³ м/с²
- **J₂-компонента:** до 90% от полного ускорения
- **Высшие гармоники:** 10⁻⁷ ... 10⁻⁵ м/с²

### Зависимость от высоты

С увеличением высоты возмущения быстро убывают:

```
|a| ~ (R₀/r)ⁿ⁺²
```

- На высоте 300 км: ~10⁻³ м/с²
- На высоте 1000 км: ~10⁻⁴ м/с²
- На высоте 36000 км (ГСО): ~10⁻⁷ м/с²

### Влияние на орбитальные элементы

**J₂ вызывает:**
- Прецессию восходящего узла Ω
- Вращение аргумента перигея ω
- Периодические изменения наклонения i

**Высшие гармоники вызывают:**
- Долгопериодические колебания элементов
- Резонансные эффекты на определённых высотах

### Анализ компонент S, T, W

**Радиальная (S):**
- Наибольшая по величине
- Влияет на форму орбиты (a, e)
- Максимальна в перигее

**Трансверсальная (T):**
- Направлена вдоль орбиты
- Изменяет скорость спутника
- Влияет на период обращения

**Бинормальная (W):**
- Перпендикулярна плоскости орбиты
- Изменяет ориентацию орбиты (Ω, i)
- Особенно важна для солнечно-синхронных орбит

### Сравнение моделей J₂-only vs Full

**Модель «J₂-only»:**
- Учитывает только основную гармонику
- Быстрее в 10-20 раз
- Точность: ~90-95% для низких орбит
- Подходит для быстрой оценки

**Полная модель:**
- Учитывает гармоники до 21-го порядка
- Высокая точность (~99%)
- Необходима для прецизионных расчётов
- Важна для специальных орбит (ГСО, резонансные)

### Практическое применение

Результаты расчётов используются для:

1. **Прогнозирования эволюции орбиты** — определение срока активного существования КА
2. **Выбора солнечно-синхронных орбит** — компенсация прецессии от J₂
3. **Коррекции орбиты** — планирование манёвров
4. **Анализа долгопериодических возмущений** — резонансы, стабилизация

### Пример интерпретации

Если получено:
- S = -5.2×10⁻⁴ м/с²
- T = 1.3×10⁻⁵ м/с²
- W = -2.1×10⁻⁵ м/с²
- |a| = 5.23×10⁻⁴ м/с²

**Выводы:**
- Доминирует радиальная компонента (99.4%)
- Основное влияние — на большую полуось и эксцентриситет
- Бинормальная составляющая вызывает медленную прецессию узла
- Типичное значение для орбиты высотой ~500 км

## Заключение

Модуль геопотенциальных возмущений предоставляет мощный инструмент для анализа влияния нецентральности гравитационного поля Земли на движение спутников. Правильная интерпретация результатов позволяет оптимизировать орбиты космических аппаратов и прогнозировать их эволюцию.
