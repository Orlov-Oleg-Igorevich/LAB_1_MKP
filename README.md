# Проект: Моделирование возмущений орбиты спутника

## Общее описание проекта

Данный проект представляет собой веб-приложение для моделирования и анализа возмущений орбиты искусственного спутника Земли, вызванных двумя основными факторами:

1. **Нецентральностью гравитационного поля Земли** (геопотенциальные возмущения)
2. **Гравитационным влиянием Луны** (лунные возмущения)

Проект разработан в рамках учебного курса МАИ (кафедра 604) и предназначен для выполнения лабораторных работ по исследованию динамики орбитального движения.

## Назначение

Приложение позволяет:

- Выполнять расчёт возмущающих ускорений от сферических гармоник геопотенциала
- Исследовать влияние лунных возмущений на орбитальное движение
- Проводить сравнительный анализ различных моделей возмущений
- Визуализировать результаты расчётов в виде графиков и 3D-моделей
- Анализировать эволюцию орбитальных элементов во времени

## Основные модули

### 1. Модуль геопотенциальных возмущений (`geopotential-module`)

Предназначен для расчёта возмущающих ускорений, вызванных нецентральностью гравитационного поля Земли.

**Ключевые возможности:**
- Разложение гравитационного потенциала в ряд по сферическим функциям
- Учёт зональных гармоник (J₂, J₃, ...)
- Учёт тессеральных и секториальных гармоник (Cₙₖ, Sₙₖ)
- Расчёт компонент ускорения в сферической системе координат (S, T, W)
- Сравнение моделей: только J₂ vs полная модель
- Работа в системах координат ECI/ECEF

**Основные файлы:**
- `apps/api/src/modules/geopotential/services/geopotential.service.ts`
- `apps/api/src/modules/calculation/services/geopotential-math.service.ts`
- `apps/api/src/modules/calculation/services/legendre.service.ts`

### 2. Модуль лунных возмущений (`lunar-perturbations-module`)

Моделирует влияние гравитационного поля Луны на орбиту спутника.

**Ключевые возможности:**
- Численное интегрирование дифференциальных уравнений движения
- Метод Рунге-Кутты-Фельдберга 4(5) порядка (RKF45)
- Адаптивный выбор шага интегрирования
- Расчёт положения Луны по элементам орбиты
- Анализ эволюции орбитальных элементов
- Сравнение возмущённой и кеплеровской орбит

**Основные файлы:**
- `apps/api/src/modules/lunar/services/lunar.service.ts`
- `apps/api/src/modules/lunar/services/lunar-analysis.service.ts`

## Пользовательский интерфейс

### Геопотенциальный модуль включает вкладки:
- **Зависимость от высоты** — графики ускорений от высоты орбиты
- **От положения на орбите** — зависимость от истинной аномалии
- **3D орбита** — визуализация орбиты в пространстве
- **Справка** — теоретическая информация

### Лунный модуль включает вкладки:
- **Элементы орбиты** — эволюция орбитальных элементов
- **Возмущающие ускорения** — компоненты S, T, W
- **Графики** — зависимости параметров
- **Сравнение** — возмущённая vs невозмущённая орбита
- **Статистика** — численные характеристики
- **3D орбита** — визуализация с положением Луны
- **Справка** — методические указания

## Физические основы

### Используемые константы

**Земля:**
- μ = 398600.4418 км³/с² — геоцентрическая гравитационная постоянная
- R₀ = 6378.137 км — экваториальный радиус Земли
- ωₑ = 7.292115×10⁻⁵ рад/с — угловая скорость вращения

**Луна:**
- μₗ = 4902.8 км³/с² — гравитационный параметр Луны
- aₗ = 384399 км — большая полуось орбиты Луны
- eₗ = 0.0549 — эксцентриситет орбиты Луны
- iₗ = 5.145° — наклонение орбиты Луны

### Системы координат

- **ECI (Earth-Centered Inertial)** — инерциальная система, связанная с Землёй
- **ECEF (Earth-Centered Earth-Fixed)** — вращающаяся система, связанная с поверхностью Земли
- **Сферическая система** — (r, φ, λ) для расчёта геопотенциала
